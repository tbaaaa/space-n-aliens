# Space and Aliens!
This is a game that will be made as a 2d spaceship alien shooter, made entirley in python.

## Guide

### Prerequisites
- Python 3.x
- Pygame library

### Installation
1. Clone the repository: `git clone https://github.com/tbaaaa/space-n-aliens.git`
2. Navigate to the project directory: `cd space-n-aliens`
3. Install dependencies: `pip install pygame`

### Running the Game
Run the game with: `python main.py`

### Controls
- Left/Right Arrow Keys: Move the spaceship
- Spacebar: Shoot bullets
- Shoot the red aliens before they reach the bottom!

### Game Objective
- Avoid letting aliens reach the bottom of the screen.
- Score points by shooting aliens.
- Game ends when an alien reaches the bottom.
