# Space N Aliens

Fight back against the alien invasion! Battle through waves of enemies and defeat powerful bosses to reclaim your galaxy.

## Installation

### Option 1: Download Executable (Recommended)
Download and play without installing python and dependencies

### Option 2: Run from Source (Tweak the src code for fun)
1. Clone: `git clone https://github.com/tbaaaa/space-n-aliens.git`
2. Install: `pip install pygame`
3. Run: `python main.py`

## Controls

- **WASD**: Move ship
- **Mouse + Left Click**: Aim and shoot
- **ESC**: Pause
- **E**: Toggle invincibility (debug)

## Gameplay

### Enemies
- **Basic Aliens**: 4 types with different movement patterns and shapes
- **Side Turrets** (Orange): Unlock after Boss 1 at score 100+
- **Horizontal Turrets** (Cyan): Unlock after Boss 2 at score 200+

### Bosses
- **Boss 1** (Score 100): Rectangular boss with spread shots
- **Boss 2** (Score 200): Star boss with reflectable projectiles
- **Final Boss** (Score 300+): Multi-stage alien with 4 attack phases

### Features
- 3 HP system with color-coded feedback
- Dynamic space backgrounds that evolve with progression
- Hyperdrive cutscenes after each boss defeat
- Scrolling starfield and distant planets
- Multiple enemy and boss attack patterns

## Objective

Survive, defeat all bosses, and achieve the highest score possible!
